# Ultimate AI Companion – Preconfigured Demo

## Features
- Fully interactive frontend dashboard
- Backend with Stripe demo checkout and Firebase auth demo
- AI workflow placeholders preloaded
- Ready for Vercel, Render, or Netlify deployment

## How to Use (Plug-and-Play)
1. Deploy the ZIP folder to **Vercel** or **Render**
2. Open the live URL → demo login/signup works automatically
3. Click any module → see deterministic demo output
4. Stripe checkout works in demo mode
5. To go live: replace Stripe test keys and Firebase config with your real credentials
